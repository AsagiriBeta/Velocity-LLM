# 示例服务器玩法指南

> 将此文件替换或复制为你自己的服务器文档。支持 .md 和 .txt 格式。

## 服务器简介

欢迎来到示例生存服务器！本服务器为群组服架构，玩家可在多个子服之间自由切换。

## 基础指令

- `/spawn` — 回到主城
- `/tpa <玩家>` — 请求传送到其他玩家身边
- `/home` — 回到个人家园
- `/money` — 查看余额

## 经济系统

玩家可通过以下方式赚取金币：

1. 出售物品给 NPC 商店
2. 完成每日任务
3. 参与服务器活动

## 领地保护

使用木斧左键选点、右键选点，然后输入 `/res create <名称>` 创建领地。领地内其他玩家无法破坏方块。

## 常见问题

**问：如何切换子服？**
答：使用 `/server <子服名>` 或在聊天中点击玩家名旁的服务器标签。

**问：物品能在子服之间带走吗？**
答：可以，本服务器使用统一背包系统。
